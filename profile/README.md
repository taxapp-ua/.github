# taxapp-ua

Розрахунок податків для ФОП 3 групи (єдиний податок без ПДВ): автоматичний облік доходу в різних валютах за курсом НБУ на дату надходження, підрахунок єдиного податку, військового збору та ЄСВ, формування квартальної звітності й платіжок.

Ввід доходу — через Telegram-бот (пізніше WhatsApp) або веб-дашборд. Уся логіка живе в бекенді; бот і фронт спілкуються з ним по HTTP.

## Репозиторії

| Репозиторій | Призначення | Стек |
|---|---|---|
| [`taxapp-api`](https://github.com/taxapp-ua/taxapp-ua_backend) | Бекенд: моделі даних, інтеграція з НБУ, движок розрахунку податків, REST API | FastAPI, SQLAlchemy, Alembic, PostgreSQL |
| [`taxapp-web`](https://github.com/taxapp-ua/taxapp-ua_frontend) | Веб-дашборд: ввід доходу, перегляд зведення за квартал, дедлайни | React |
| [`taxapp-bot`](https://github.com/taxapp-ua/taxapp-ua_tgbot) | Месенджер-бот для вводу доходу та перегляду податків | Python, aiogram |

## Як це працює

1. Користувач вносить дохід (сума, валюта, дата) через бот або веб.
2. Бекенд тягне офіційний курс НБУ на дату надходження й перераховує дохід у гривню.
3. Движок рахує єдиний податок (5%), військовий збір (1%) та ЄСВ наростаючим підсумком за квартал.
4. Формується квартальна декларація та платіжка з реквізитами для сплати.

## Дорожня карта

- **Фаза 0** — ядро: моделі даних, сервіс курсу НБУ, движок розрахунку з тестами.
- **Фаза 1** — Telegram-бот для вводу доходу.
- **Фаза 2** — React-дашборд.
- **Фаза 3** — генерація готової декларації та платіжки (заповнений документ для ручної подачі).
- **Фаза 4** — автопідпис КЕП (ДСТУ 4145) та автоподача через API Електронного кабінету ДПС.

## Застереження

Це допоміжний інструмент, а не заміна бухгалтера. Податкові ставки, ліміти, формати декларацій і бюджетні реквізити періодично змінюються — перед подачею звітності звіряй розрахунки з актуальними правилами ДПС або з фахівцем. Особлива увага — до моменту визнання доходу при роботі через Payoneer (дата зарахування коштів на рахунок ФОП).
